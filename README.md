# WhispNote

WhispNote is a modern, minimal, and magical speech-to-text note-taking web application. It features a beautiful interface with glassmorphism and neumorphism design elements, creating an elegant and intuitive user experience.

## Features

- **Speech-to-Text Recording**: Capture your thoughts by speaking
- **Live Transcription**: See your words appear in real-time as you speak
- **Note Management**: Save, edit, and delete your notes
- **Tag Support**: Automatically extracts hashtags from your notes
- **Dark/Light Mode**: Toggle between beautiful dark and light themes
- **Responsive Design**: Works on mobile, tablet, and desktop
- **Elegant Animations**: Smooth transitions and micro-interactions
- **Local Storage**: Your notes are saved in your browser

## Getting Started

1. Clone or download this repository
2. Open `index.html` in a modern web browser (Chrome or Edge recommended for best speech recognition support)
3. Click the microphone button to start recording
4. Speak clearly, and your words will appear in the transcription area
5. Click the button again to stop recording and save your note

## Sound Effects

For the full experience, add your own sound effect files to the `sounds` directory:
- `start.mp3`: Played when recording starts
- `stop.mp3`: Played when recording stops
- `save.mp3`: Played when a note is saved

You can find free sound effects at:
- [Freesound](https://freesound.org/)
- [Mixkit](https://mixkit.co/free-sound-effects/)
- [Zapsplat](https://www.zapsplat.com/)

## Browser Compatibility

WhispNote works best in modern browsers that support the Web Speech API:
- Google Chrome (recommended)
- Microsoft Edge
- Safari (partial support)
- Firefox (partial support)

## Technologies Used

- HTML5
- CSS3 (with modern features like glassmorphism, custom properties, and animations)
- JavaScript (vanilla, no frameworks)
- Web Speech API for speech recognition

## Design Elements

- **Glassmorphism**: Transparent, frosted-glass cards and containers
- **Neumorphism**: Soft shadow effects for the microphone button
- **Dark Mode**: Dark background with neon accent colors
- **Light Mode**: Light background with soft pastel tones
- **Micro-interactions**: Subtle animations and feedback
